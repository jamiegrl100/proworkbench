## Quick Start

PB is a local-first AI workbench. For v1 preview, PB uses **Text Generation WebUI** as the local model server (OpenAI-compatible API).

### Developers / AI folks (fast path)

```bash
git clone https://github.com/jamiegrl100/proworkbench.git
cd proworkbench
npm install
npm run dev
```

Open: `http://localhost:5173`

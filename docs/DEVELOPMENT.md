# Development

## Local dev

```bash
npm install
npm run dev
```

## Typical ports

- UI: `5173`
- API: `8787`
- Model backend: `5000`

## Repo shape

- `ui/` frontend
- `server/` backend
- `docs/` documentation
- `assets/` screenshots/diagrams/brand

## Quality checks

- Run existing tests/lint (if configured)
- Keep docs + website consistent in the same PR
- Avoid unscoped feature claims

## Contribution flow

1. Branch from `main`
2. Add changes + docs
3. Open PR with validation steps
4. Request review

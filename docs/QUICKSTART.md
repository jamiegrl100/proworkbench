# Quickstart (5 minutes)

1. Start ProWorkBench:
   - `npm install`
   - `npm run dev`
2. Open UI: `http://127.0.0.1:5173`
3. Open **Doctor** and run checks.
4. Configure model endpoint (`http://127.0.0.1:5000`).
5. Send a WebChat prompt and validate response path.

## First-run validation

- Doctor: all critical checks green.
- WebChat: one direct prompt succeeds.
- MCP: one template health check succeeds.

If a check fails, continue in [TROUBLESHOOTING.md](./TROUBLESHOOTING.md).
